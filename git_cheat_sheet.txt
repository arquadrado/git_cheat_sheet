Setup Git

$ git config --global user.name "<name>"
$ git config --global user.email "<email>"

Initialize git repo
$ git init

Project status

$ git status

Add file to stage

$ git add <filename>

Commit file

$ git commit
	
	options
	-m "message" // add message to commit
	-a           // add files to stage


Check repo history
$ git log

	option
	--graph     // branch graph
	--oneline   // show in one line
	-p          // chnages
	-1          // number of commits
	--decorate 

Create alias
$ git config --global alias.lg 'log --graph --oneline --decorate --all'

See differences

$ git diff            // diff between untracked and commit

	options
	--staged   // diff between stage and commit


Reset local changes

$ git cheackout	-- <filename>   // reset local changes in file

Reset the stage

$ git reset HEAD

	options
	<filename>    // unstage file

Remove commit

$ git reset <commitID>

	options
	--hard     // remove changes

Amend last commit

$ git commit --amend
